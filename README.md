# OIBSIP - OASIS Infobyte Python Programming Internship 🐍

### 👤 Author: Anupam Ram Tripathi

Welcome to my repository for the **OASIS Infobyte Python Programming
Internship**! This repo contains all the tasks completed during the
internship, each organized in its own folder with source code and a
dedicated README.

---

## 📁 Tasks Completed

| # | Task | Description | Tech Used |
|---|------|--------------|-----------|
| 1 | [Voice Assistant](./Task1_VoiceAssistant) | A voice-controlled assistant that responds to greetings, tells time/date, opens apps, and searches the web. | Python, speech_recognition, pyttsx3/win32com |
| 2 | [Weather App](./Task2_WeatherApp) | A GUI weather application that fetches live weather data using the OpenWeatherMap API. | Python, Tkinter, OpenWeatherMap API |
| 3 | [BMI Calculator](./Task3_BMICalculator) | A GUI-based BMI calculator that computes and classifies Body Mass Index. | Python, Tkinter |
| 4 | [Random Password Generator](./Task4_PasswordGenerator) | A GUI app that generates strong, customizable random passwords with a strength indicator. | Python, Tkinter, random, string |

---

## 🛠️ How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/tripathianupam8957-cloud/OIBSIP.git
   ```
2. Navigate into any task folder:
   ```bash
   cd OIBSIP/Task1_VoiceAssistant
   ```
3. Follow the instructions in that task's own `README.md` to run it.
